 # ⚔️  KingShot Gift Code Redeemer                                                                                                                                         
                                                                                                                                                                           
  자동으로 길드원 전체에게 기프트코드를 등록해주는 데스크탑 앱이에요.                                                                                                      
  A desktop app that automatically redeems gift codes for all guild members.

  ## ✨ Features / 기능

  - 🎁 여러 기프트코드 한번에 등록 / Redeem multiple gift codes at once                                                                                                    
  - 👥 길드원 직접 입력 또는 CSV 파일로 불러오기 / Add members manually or via CSV
  - 🌙 다크/라이트 모드 / Dark & Light mode                                                                                                                                
  - 🇰🇷 🇺🇸 한국어/영어 지원 / Korean & English support                                                                                                                      
  - ✅ 성공/실패/중복수령 실시간 로그 / Real-time success/fail/skip log

  ## 🚀 사용법

  **macOS**
  1. `KingShot.GiftCode.mac.zip` 다운로드 후 압축 해제
  2. 앱 **우클릭 → 열기** (최초 1회)                                                                                                                                       
  3. 기프트코드 입력 → 길드원 추가 → 시작!
                                                                                                                                                                           
  **Windows**     
  1. `KingShot.GiftCode.windows.zip` 다운로드 후 압축 해제                                                                                                                 
  2. `KingShot GiftCode` 폴더 안에서 `KingShot GiftCode.exe` 실행
  3. 기프트코드 입력 → 길드원 추가 → 시작!                                                                                                                                 
   
  ---           
                                                                                                                                                                           
  ## 🚀 How to Use

  **macOS**
  1. Download and unzip `KingShot.GiftCode.mac.zip`
  2. **Right-click → Open** (first time only)                                                                                                                              
  3. Enter gift code → Add members → Start!
                                                                                                                                                                           
  **Windows**     
  1. Download and unzip `KingShot.GiftCode.windows.zip`                                                                                                                    
  2. Open `KingShot GiftCode` folder and run `KingShot GiftCode.exe`
  3. Enter gift code → Add members → Start!

  ## 📋 (중요!!) CSV Format / CSV 형식
                                                                                                                                                                           
  The first row must be a header. `player_id` is required, name is optional.                                                                                               
  첫 번째 줄은 헤더여야 해요. `player_id` 는 필수, 이름은 선택이에요.                                                                                                      
                                                                                                                                                                           
  ```
  player_id, name                                                                                                                                                          
  191512817, Jiye 
  190635247, Lemon
  232484716, Pikachu
  ```

  - `player_id` : required / 필수                                                                                                                                          
  - `name` : optional (if empty, player_id is used) / 선택 (비우면 ID로 표시)
